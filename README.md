# 🏋️‍♂️ GymRats - Gym Management System

**GymRats** es una solución integral para la gestión de gimnasios y control de aforo en tiempo real. Este proyecto combina una arquitectura moderna para ofrecer una experiencia fluida tanto a empresas como a deportistas.

## 📁 Estructura del Repositorio

- **/Backend**: Servidor API REST desarrollado con Python y FastAPI.
- **/ScriptsSQL**: Colección de scripts para la creación y migración de la base de datos en Supabase.

## 🚀 Tecnologías Principales

- **Backend:** Python 3.10+, FastAPI, Uvicorn.
- **Base de Datos:** Supabase (PostgreSQL).
