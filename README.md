# 🏋️‍♂️ GymRats - Gym Management System

**GymRats** es una solución integral para la gestión de gimnasios y control de aforo en tiempo real. Este proyecto combina una arquitectura moderna para ofrecer una experiencia fluida tanto a empresas como a deportistas.

## 🌐 API y Documentación en Vivo

El backend se encuentra desplegado y puede ser testeado de forma interactiva en:
👉 **[GymRats API Dashboard (Swagger UI)](https://gymrats-api-jrrw.onrender.com/docs)**
*Nota: debido al uso del plan gratuito de [render.com](https://render.com/) es posible que se deba esperar un poco antes de poder hacer uso de la API.*

## 📁 Estructura del Repositorio

* **/Backend**: Servidor API REST asíncrono desarrollado con Python. Contiene la lógica de negocio, autenticación JWT y gestión de base de datos.
    * 📖 *Para guías de instalación, requisitos y uso detallado, consulta el [README de Backend](./Backend/README.md).*
* **/ScriptsSQL**: Repositorio de esquemas relacionales, triggers y funciones almacenadas para PostgreSQL (Supabase).
